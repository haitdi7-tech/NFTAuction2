# 项目结构说明

## 项目目录结构

```
NFTAUCTION2/
├── contracts/              # 智能合约
│   ├── AggreagatorV3.sol             # 喂价合约
│   ├── MyERC20.sol                   # ERC20代币合约
│   ├── MyNFT.sol                     # NFT合约
│   ├── MyAuction.sol      # 可升级拍卖合约
│   └── NFTAuctionV2.sol   # 升级后的拍卖合约
|
├── deploy/                 # 部署脚本
│   ├── 01-deploy-NFTAution.js
│   └── 02-upgrade-NFTAution.js
├── test/                   # 测试脚本
│   └── test.js
├── hardhat.config.js       # Hardhat配置
├── .env                    # 环境变量
└── package.json            # 依赖配置
└── 项目文档.md              # 项目结构
```

---
## 功能说明

### 核心功能

1.NFT铸造: 支持ERC721标准NFT的创建和转移

2.拍卖机制: 支持固定时长拍卖，最高价获胜

3.多币种出价: 支持ETH和ERC20代币出价（V2）

4.价格转换: 通过Chainlink预言机实时计算美元价值

5.合约升级: 使用UUPS代理模式实现无缝升级

## 部署步骤

```bash
# 1. 部署
npx hardhat deploy --tags NFTAuction

# 2. 测试
npx hardhat test 
```
